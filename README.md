# DigiSchool

An Edtech platform for teachers, pupils, administrators and parents. Each will have their own frontend. It should be able to scale up to be used in as a main tool for teachers to smoothly integrate physical and online classrooms into on space. 

# Architecture

***Users***
* Students
* Teachers
* Parents
* Admin

***Clients***
* Learning
* Teaching
* Student overview
* Admin

***Internal***
* Auth
* Gateway
* ClassroomService
* MessagingService[websocket]
* CourseService
* AssessmentService
* CollaborationService[websocket]
* UserAccount
* Scheduling
